# Koby 升级优化记录

> 记录应用的功能升级和优化进展

## 已完成

### 核心功能

- [x] **i18n 国际化** — 全站中英文双语支持，包含表单占位符国际化
- [x] **无限滚动** — 替换分页为 IntersectionObserver 懒加载（AllBookmarksView、CollectionsView）
- [x] **全局搜索** — Ctrl+K / Cmd+K 快捷键打开搜索弹窗，即时客户端搜索 + 后端搜索双模式
- [x] **URL 元数据自动填充** — 新增书签时自动获取标题、描述、favicon
- [x] **收藏夹树形结构** — 支持收藏夹嵌套管理和层级下拉选择
- [x] **Toast 通知系统** — 操作反馈通知（添加/编辑/删除成功/失败）
- [x] **防重复提交** — 表单操作加锁防止重复提交
- [x] **nanoid 唯一 ID** — 使用 nanoid 生成唯一 ID 替代自增整数
- [x] **书签分享** — 一键复制书签标题和链接到剪贴板（所有书签卡片均支持）
- [x] **URL 重复检测** — 新增书签时自动检测 URL 是否已存在，显示所在收藏夹提示

### UI/UX 优化

- [x] **书签卡片紧凑设计** — 移除大尺寸 banner，改为 favicon + 内容水平布局
- [x] **标题可点击跳转** — 书签标题直接链接到目标网址，URL 改为纯文本展示
- [x] **收藏夹颜色应用** — 卡片 icon 背景色、收藏夹标签背景色使用收藏夹自定义颜色
- [x] **标签展示差异化** — 收藏夹标签使用彩色背景，书签标签使用 #tag 文本样式
- [x] **收藏夹下拉层级显示** — 所有收藏夹选择器支持缩进层级展示
- [x] **新增书签默认收藏夹** — 从收藏夹详情页添加书签时自动选中当前收藏夹
- [x] **首页数据限制** — 收藏夹限制显示 10 条，最近添加显示 8 条
- [x] **卡片悬停动效** — 优化卡片 hover 过渡效果
- [x] **搜索按钮快捷键提示** — 顶栏搜索按钮显示 Ctrl+K / Cmd+K 快捷键提示
- [x] **收藏夹书签卡片优化** — 卡片间距和视觉层次优化，内容更宽松易读
- [x] **首页置顶书签模块** — 首页新增置顶书签区域，琥珀色边框突出显示，支持取消置顶

### 性能优化

- [x] **后端搜索** — 新增 `/api/bookmarks/search` 接口，SQL LIKE 查询，SearchModal 支持双模式（即时客户端 + 延迟后端）
- [x] **图片懒加载** — 所有 favicon `<img>` 标签添加 `loading="lazy"` 属性
- [x] **渲染优化** — 书签卡片使用 CSS `content-visibility: auto` 跳过屏外渲染
- [x] **Service Worker / PWA** — 使用 vite-plugin-pwa 生成 Service Worker，支持离线缓存静态资源和图片

### 基础设施

- [x] **项目文档** — CC_FOR_CHENYUN.md 项目全面说明文档
- [x] **README 更新** — 更新功能列表和项目结构说明
